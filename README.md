# Smart Campus Resource Management System

## Project Description
This project is a Smart Campus Resource Management System that allows users to manage and book campus resources like classrooms and labs.

## Features
- User Registration 
- User Login system
- Show available resources (Classrooms & Labs)
- Book resources
- Check availability
- Cancel bookings
- Show all resources
- Show MST (Minimum Spanning Tree using algorithms)

## Technologies Used
- C++ (Backend Logic)
- HTML, CSS, JavaScript (Frontend)
- MySQL (Database)

## Project Structure
- Frontend files (HTML, CSS, JS)
- Backend logic (C++)
- Database (MySQL)

## Algorithms Used
- Greedy Algorithm
- Minimum Spanning Tree (Prim's/Kruskal's Algorithm)

## Purpose
This project helps in efficient management of campus resources and avoids conflicts in booking.
